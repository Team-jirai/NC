# Note:green_book:
 
会員(Customer)
```bash
・ 商品閲覧
・ カテゴリ検索
・ 商品の購入
・ 注文履歴の閲覧
・ 登録情報の変更
```

管理者(Admin)
```bash
・ topページに当日の注文件数表示
・ 全会員の表示
・ 送料・支払方法の設定←new!
・ 商品ジャンル・商品の追加可能
・ 全会員の注文履歴の閲覧
```
